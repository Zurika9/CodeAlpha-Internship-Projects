# CodeAlpha Internship Projects

This repository contains all the projects completed as part of my CodeAlpha Internship program. It includes multiple Java-based applications developed to strengthen core programming, object-oriented design, and problem-solving skills.

## 📁 Projects Included

### 1. Student Grade Tracker
A Java-based application to manage student records, calculate grades, and generate performance summaries using OOP principles.

### 2. Stock Trading Platform Simulation
A console-based simulation of a stock trading system that allows users to buy/sell stocks and track portfolio performance.

### 3. AI Chatbot
An interactive Java-based chatbot that responds to user queries using rule-based logic and basic NLP techniques.

## 🛠️ Tech Stack
- Java
- Object-Oriented Programming (OOP)
- File Handling (if applicable)
- Basic Data Structures

## 🎯 Purpose
The goal of these projects is to demonstrate practical implementation of Java programming concepts, improve logical thinking, and gain real-world development experience through structured internship tasks.
