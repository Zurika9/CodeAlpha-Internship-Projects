# 📈 Stock Trading Platform

> **CodeAlpha Java Programming Internship – Task 2**

A professionally designed **Java console-based Stock Trading Platform** that simulates stock market trading. The application allows users to buy and sell stocks, manage an investment portfolio, monitor portfolio performance, maintain transaction history, and persist data using file handling.

This project demonstrates the practical implementation of **Object-Oriented Programming (OOP)** principles, exception handling, collections, file I/O, modular software design, and clean console-based user interfaces.

# ✨ Features

### 📊 Stock Market

- View available stocks
- Display company names
- Display current stock prices
- Simulate market price fluctuations

---

### 💹 Trading

- Buy stocks
- Sell stocks
- Input validation
- Insufficient balance checking
- Invalid stock handling

---

### 📁 Portfolio Management

- View owned stocks
- View quantity of shares
- Calculate current portfolio value
- Display investment summary

---

### 📈 Portfolio Performance

- Current portfolio valuation
- Total investment
- Profit/Loss calculation

---

### 📝 Transaction History

- Record every BUY transaction
- Record every SELL transaction
- Store transaction date & time
- Display formatted transaction history

---

### 💾 Data Persistence

Application automatically saves:

- User Information
- Portfolio
- Transactions
